# Chicago Motor Vehicle Theft Analysis (R)

## Objective
Analyze motor vehicle theft patterns in Chicago using R, focusing on
temporal trends, arrest rates, and high-risk locations.

## Key Concepts
- Reading and exploring large datasets
- Working with date-time variables
- Subsetting and filtering data
- Visualizing trends with histograms and boxplots
- Using tables and proportions for comparison

## Key Insights
- Motor vehicle thefts declined over time
- Arrest rates were higher in earlier years
- Gas stations show higher arrest rates
- Location and timing matter for enforcement strategy
